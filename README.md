# student-management-system

A console-based **Student Management System** developed in **C++**. This project allows users to manage student records using a simple menu-driven interface and demonstrates core Object-Oriented Programming (OOP) concepts, the Standard Template Library (STL), and file handling.

## Features

* Add a new student
* View all student records
* Search for a student by Roll Number
* Update student information
* Delete a student record
* Save student data to a file
* Load student data when the program starts

> **Note:** Some of these features are planned and will be implemented as the project progresses.

## Technologies Used

* Object-Oriented Programming (OOP)
